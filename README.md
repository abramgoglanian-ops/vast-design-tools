# Vast Design Tools

Interactive design system tools, references, and explorers for the Vast design team.

## What lives here
- Interactive HTML tools (color explorers, token references, design utilities)
- Self-contained, non-sensitive design aids

## What does NOT belong here
- Internal briefs, specs, or strategic documents
- Unreleased product information
- Partner, vendor, or customer data
- Any content that may be ITAR-controlled or export-restricted
